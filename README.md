## Library-Management
This is spring boot micro-service project to mange all operation for library

#### Tools and Technologies
- Spring Boot- 2.1.5
- REST API Standard
- Lombok
- H2 Database 
- Logging
- Javax validation
- Junit 5
- ModelMapper , DTO
- httpclient

#### Instruction to run :
    Build mvn clean install 
    Execute command to run the project: 
        mvn spring-boot:run

URI -
http://localhost:9090/api/library/books
